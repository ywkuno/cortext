# brain-viz (future CodePrism app)

This folder is reserved for a future polished visualization app.

Recommended stack:
- React
- TypeScript
- Vite
- Cytoscape.js or Sigma.js

Why keep this app separate from the Python CLI?
- cleaner UX iteration
- easier frontend polish
- can become a standalone desktop/web companion
- can eventually support live reload and activity stream replay
