# Context Pack Format

A context pack is a compact project brief intended for AI tools.

## Markdown sections

1. Scan metadata
2. Project structure
3. Important files
4. Symbols
5. Imports / edges
6. Documentation headings
7. Suggested files to read next
8. Agent notes

## Ranking ideas

A file becomes important when it is an entrypoint, imported by many files, a config file, a route/controller, recently changed, mentioned in docs, or requested by the user query.
