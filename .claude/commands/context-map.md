# /context-map

Map this repository for efficient AI context use.

Run:

```bash
codeprism init
codeprism map .
codeprism stats
codeprism gain
codeprism mcp --list-tools
```

Then summarize project shape, likely entrypoints, parser coverage, map freshness, estimated savings, available MCP tools, and the best query to slice next. Avoid broad raw file reads unless the map is insufficient.
